Javacript basic

Javascript is a high-level programming language and designed to run in webbrowsers. It can be used to manipulate webpage-content and respond to user event. Javascript is often used with html and css to create dynamic and engaging webpages.

This can lead to confusion and unintended consecuenses, especially when more than one developer is working on the code. Therefor its not good practice.

Primitive Types:
  • number: Integer and float
  • string: sequence of characters
  • boolean: logical value, true or false
  • null: non value or empty value
  • undefined: variable that has been assigned value
  • symbol
Object Types:
  • object: collection of properties and methods
  • function: reusable code that performs a task
  • array: collection of values(any type)

Variables are declared not with a specific datatype, but instead the type is determined dynamically based on the value that is assigned
