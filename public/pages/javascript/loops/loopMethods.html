Loop Methods

In javascript there are several loop methods available in javascript that makes it possible to quickly iterate over arrays.

forloop
- which we used in java from 1 semester. It goes through all elements in that array.

for (let i = 0; i < array.length; i++) { console.log(array[i]);}

foreach
- iterate over each item and can apply a function to each of them

array.forEach(item => { console.log(item); });

map()
- iterate over an array and creates a new array by executing the function between the brackets for each element of the original array.

spreadoperator is here used to create a new object with all the properties of the original array. This is to ensure that the original array remains unchanged. If i added the owner property and didnt use the spreadoperator, it would change the original array as well and we would have unintended sideeffects.(effects that weren't intended)

        
//iterate though spaceships array and for each element in that array it
    sets the propery "owner" to be "china". Instead of return just the new array.
    We use a spreadoperator to we avoid mutuble arraylist.

const spaceshipWithOwners = spaceships.map((spaceship) =>{
    spaceship.owner = "china"
    return {...spaceship, owner: "china"};
});

filter()
- is used to create a new array with all elements from that array that has passed through a specific test. Basicly we filter the array by giving it a condition to filter from.

        
Basicly it makes a new array with all the elements that has the property priceless equal to true
const newSpaceshipArray = spaceships.filter(spaceship => spaceship.isPriceless === true)

reduce()
- is used to reduce an array to a single value by executing a function for each element in that array. It takes two params; a callbackfunction and an initial value, where the accumulation should start.

It calls the callbackfunction on every element and returns acumulated result of that.

        
//reduce() counts the totalprice of the spaceships
//accumulator holds the running value of 
const totalPrice = spaceships.reduce((accumulator, spaceship) =>{
    return accumulator + spaceship.price;
}, 0)

find()
- returns the value of the first element where the condition is met (always a single value) or undefined.

//It finds the spaceship that has the property name called cam. const specificSpaceship = spaceships.find(spaceship => spaceship.name === cam)
