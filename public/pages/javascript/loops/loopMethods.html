Loop Methods

In javascript there are several loop methods available in javascript that makes it possible to quickly iterate over arrays.

forloop which we used in java from 1 semester. It goes through all elements in that array.

        
for (let i = 0; i < array.length; i++) {
    console.log(array[i]);
}

foreach iterate over each item and can apply a function to each of them

        
array.forEach(item => {
    console.log(item);
});
