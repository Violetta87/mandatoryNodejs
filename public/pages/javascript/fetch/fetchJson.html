JSON format

JSON (Javascript Object Notation) is a standard text-based format. It is represented in key value pairs. The key is always a string, but the value can be a string, number, boolean, array. It commonly used for transmitting data in webapplications. Javascript has a global JSON object that has methods available to convert between JSON and javascripts objects.

        
{
    "name": "Cam",
    "age": 35,
    "Hobbies": ["music, fitness", "programming", "travels"]
}
