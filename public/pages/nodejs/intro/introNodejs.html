Introduction to Nodejs

Nodejs is an open source, cross-platform. server-side runtime enviroment buildt on Chrome's V8 engine. Basicly what it does is that it enables you to run javascript code on the server, instead of just in the browser. Nodejs makes it possible to make a fullstack application, backend and frontend in the same language; javascript.

It runs on the serversite and provides an eventdriven and a non-blocking I/O model(input/out). This means that instead of waiting for input and output operations to complete and before moving to the next operation, Nodejs uses an event loop to process multiple I/O operations asynchronously. This makes it possible for Nodejs to handle a large number of I/O operations a the same time without blocking the execution of the applikation. Nodejs is therefor a suitable tool for making scalable and high performance fullstack applikations.

I/O model determines how input from clients(http requests, websocket messages) is being processed. I/O Model is important for serverside and clientside implementation.
