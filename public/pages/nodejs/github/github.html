How to quickly connect to Github

Github is a web-based platform that gives developers a way to store and manage their code repositories. It is a git based version control system that allows multiple developers to work on the same repository simultaneously. Github provides different sorts of tools like pull request, code review, branches, issue tracking to help teams coordinate and manage their software projects. Github is widely used around the world by individuels, start ups and large companies.
