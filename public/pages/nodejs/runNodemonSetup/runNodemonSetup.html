How to run nodejs Projects?

The way we run our nodejs application is by running the main module.

Make notes that its important to "cd" into the project you want to run. When you are in the given directory. Run this command with the main module:

            
node app.js

But the command above means that you have to start the server everytime you made changes. - By installing nodemon. You only have to start the server one time and it automaticly starts the server when ever changes has been made.
