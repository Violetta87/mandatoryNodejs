How to run nodejs Projects?

The way we run our nodejs application is by running the main module; app.js which is typically our application entry point(best practices). But its possible to run any of the modules in the nodejs project.

Make notes that its important to "cd" into the project you want to run. When you are in the given directory. Run this command in the terminal: (could be either in VS terminal or any other terminal in the computer.)

node app.js

But the command above means that you have to start the server everytime you make changes. By installing nodemon you only have to start the server one time and it automaticly starts it when ever changes has been made.

The installation of Nodemon:

1. Use Node package manager og and type:

//-g stands for globally on the machine.(installed not just in the project) npm install -g nodemon

Once we have installed nodemon. We just need to start the server one time in the terminal, and it updates automaticly.

//this have to be typed to start the nodemon in my computer due to my computers operating system - but in other pc just nodemon should work. nodemon.cmd app.js

However nodemon monitors the serverside and does not monitor if changes are made in the frontend. We can resolve this problem by adding a "nodemon.json" file containing this:

//The property "ext" stands for extention with the values: html, css, js - which makes nodemon monitor those files for changes too. { "ext": "html, js, css" }
