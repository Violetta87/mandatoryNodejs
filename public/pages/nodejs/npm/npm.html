Node Package Manager

Difference between Maven and NPM

Both Maven and NPM are package managers. A package manager is a software tool which is used to manage the installation, updating and removal of software packages or modules. They both do the same, but in different ways. Maven manages packages for java developement and uses a more complex configuation system based om XML files including POM.XL which includes dependencies and information about the project.

Npm only uses one configuration file which is called package.json. When ever a package is installed by using npm , the package and its dependencies are downloaded and installed into node_modules

How to install

One way to install dependencies

1. Make a file in your project called package.json

2. In the package.json file. You can manually write the dependencies to be installed eks.:

{ "dependencies": "life-cycles" }

by writing this in the terminal it installs all the dependencies in package.json:

npm install
Another way to do it is:
npm install life-cycles

With that command It automaticly makes the package.json file and it installs the packages and its dependencies in node_modules

We can always uninstall packages in the terminal by writing this

npm uninstall life-cycles
